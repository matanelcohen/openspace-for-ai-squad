Built on top of

📦 Squad SDK by Brady Gaster

Squad gives you the CLI and SDK to create AI agent teams with squad init and squad build. openspace.ai enhances it with a visual dashboard — letting you manage multiple squad workspaces, see what agents are doing in real-time, search across all project knowledge, and configure skills and memories through a UI instead of editing files.

npm i @bradygaster/squad-sdk → Squad GitHub →

What You Can Do

A full workspace for managing AI agent teams — from hiring to shipping.

🏗️

Create a Squad in Seconds

Describe your project, pick a theme (Marvel, Diablo, Ocean's Eleven), and the LLM generates a team with roles, personalities, and full charters — using the Squad SDK under the hood.

📋

Create & Assign Tasks

Drag-and-drop task board with backlog, in-progress, review, and done. Assign tasks to agents — they pick them up, execute using Copilot, and report results. Track progress in real-time.

💬

Chat with Your Team

Talk to the whole squad or DM individual agents. ChatGPT-style interface with agent avatars, typing indicators, and conversation memory. Agents respond in character.

👥

Manage Team Members

View each agent's charter, skills, and work history. Edit charters, toggle skill overrides (auto/always/never), and see what each agent has learned from past tasks.

🔄

Switch Between Projects

Manage multiple squad workspaces — each with its own team, tasks, chat, and knowledge base. Switch instantly. Add new projects by pointing to any folder with a .squad/ directory.

📊

Full AI Observability

Every LLM call is traced: prompt, response, model, tokens, cost, duration, and which skills were used. Debug agent behavior and optimize performance.

🧠

Agents That Learn

Agents build memory from completed tasks and project history. Relevant memories are retrieved via vector similarity and injected into context — so agents don't repeat mistakes.

🔍

Search Everything

Knowledge base indexes docs, tasks, git commits, PRs, chat threads, and agent memories. Semantic search with local embeddings — no API keys needed.

⚡

Smart Skill Matching

29 built-in skills from the Squad SDK template library. Skills are matched to tasks automatically by content + role + domain — or pinned per agent as overrides.
