Everything You Need

A complete platform for managing AI agent teams with real-time collaboration.

🎭

Themed Agent Teams

Create squads with themed characters — Marvel heroes, Ocean's Eleven, Diablo — or let the LLM generate custom themes. Each agent gets a personality and backstory.

🧠

Agent Memory

Agents learn from past work. Memories are synced from history.md, scored by vector similarity, and injected into context for relevant tasks.

⚡

SDK-Powered Skills

Skills are matched dynamically per task using keyword + role + domain scoring. Override with Always/Never per agent. Tracked in traces.

💬

Real-time Chat

ChatGPT-style chat with agent avatars, typing indicators, and channel-based routing. Messages scoped per workspace.

📊

AI Traces

Full observability: every LLM call traced with prompt, response, model, tokens, skills used, and duration. OTel-compatible.

🔍

Knowledge Base

RAG pipeline indexes docs, tasks, commits, PRs, chat, memories, and charters. Local embeddings with all-MiniLM-L6-v2 for semantic search.

📋

Rich Charters

Generated from Squad SDK's role catalog with identity, expertise, boundaries, collaboration rules, and model preferences. Full useRole() integration.

🔄

Multi-Workspace

Switch between projects instantly. Each workspace has its own team, chat, traces, skills, and knowledge — fully isolated.

⏰

Cron & Ceremonies

Schedule standups, retros, and custom ceremonies. Full CRUD with cron expression preview and execution history.
