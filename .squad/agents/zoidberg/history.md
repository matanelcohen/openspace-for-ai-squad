Playwright installed at workspace root with `pnpm e2e` script, `playwright.config.ts` configured for Chromium with dual webServer (API + web), and `e2e/smoke.spec.ts` covering app shell, navigation, theme toggle, and task board routes. All 648 project tests pass (253 web + 395 api).
