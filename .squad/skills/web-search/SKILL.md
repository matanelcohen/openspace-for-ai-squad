## Web Search

You can search the web for up-to-date information about libraries, APIs, frameworks, and best practices.

- **Documentation** — find official docs for packages and APIs
- **Examples** — find code examples and patterns
- **Troubleshooting** — research error messages and known issues
- **Best practices** — find current recommendations for tools and frameworks

### Guidelines

- Formulate specific, targeted search queries
- Prefer official documentation over blog posts
- Verify information from multiple sources when possible
- Cite sources when providing information from search results
