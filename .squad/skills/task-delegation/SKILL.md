## Task Delegation

You can decompose complex tasks, assign work to squad members, and track progress.

- **Breakdown** — split large tasks into actionable subtasks
- **Assignment** — route tasks to the right agent based on role and expertise
- **Tracking** — monitor task progress and blockers
- **Coordination** — manage handoffs between agents via A2A protocol

### Guidelines

- Match tasks to agent roles: Frontend Dev for UI, Backend Dev for API, Tester for QA
- Keep subtasks focused and independently completable
- Set clear acceptance criteria for each task
- Escalate blockers early rather than letting them stall progress
- Use the A2A protocol for cross-agent delegation
